# CannonPackingAPI
Es una RESTful API diseńada para gestionar y optimizar el embalaje de artículos para su transporte.  
Proporciona endpoints para crear items y cajas, ver su lista, eliminar de forma suave (soft deleting) y empacar productos.

## :wrench: Tecnologías utilizadas
CannonPackingAPI está construida con tecnologia de .NET 9.0, utilizando C# como lenguaje de programación.  
La API sigue el patrón de diseńo RESTful y se implementa utilizando ASP.NET Core Web API y se ejecuta usando la tecnologia Swagger.
